## 🚀 ویژگی‌ها

- **پشتیبانی از چندین پروتکل:** ایجاد و مدیریت کاملاً خودکار سرورهای SSH مستقیم و V2Ray (VLESS Reality).
- **ادغام با ربات تلگرام:** ربات پیشرفته تلگرامی ساخته شده با `aiogram` برای تعامل کاربر، خرید پلن و مدیریت اشتراک.
- **مینی اپلیکیشن داخلی (Mini App):** یک WebApp مدرن تلگرامی برای تجربه کاربری جذاب، که به کاربران امکان مدیریت سرویس‌ها بدون خروج از تلگرام را می‌دهد.
- **سیستم پرداخت خودکار:**
  - **پرداخت رمزارزی:** درگاه کاملاً خودکار کریپتو با استفاده از **NOWPayments API**.
  - **پرداخت ریالی (کارت به کارت):** جریان تأیید دستی رسیدهای پرداخت از طریق یک گروه مدیریت ادمین یکپارچه با ربات.
- **داشبورد مدیریت پیشرفته:** مشاهده آمار لحظه‌ای، مدیریت کاربران، و سیستم تأیید پرداخت سریع.
- **مسیریابی هوشمند و عبور از فیلترینگ:** طراحی شده به‌صورت ویژه برای محیط‌های با سانسور شدید جهت تضمین اتصالی پایدار و امن (از جمله اینترنت ملی).
- **ساختار مبتنی بر داکر (Dockerized):** کانتینرهای آماده برای استقرار سریع PostgreSQL، Redis، FastAPI، و سرور SSH.

## 🏗 معماری پروژه

پروژه به چندین بخش قدرتمند تقسیم شده است:

- **بک‌اند FastAPI (`/app/api/`)**: مدیریت مسیرهای WebApp، وب‌هوک‌های NOWPayments، و رابط‌های کاربری (REST API).
- **ربات تلگرام (`/app/bot/`)**: پردازش تمام دستورات تلگرام، دکمه‌های شیشه‌ای، و سیستم تأیید فیش‌های واریزی.
- **سرویس‌ها (`/app/services/`)**: هسته اصلی منطق برنامه شامل مدیریت کاربران SSH، ارتباط با پنل V2Ray، و جریان پرداخت‌ها.
- **پایگاه داده (`/app/db/`)**: پایگاه داده PostgreSQL به‌صورت Asynchronous با استفاده از SQLAlchemy ORM.
- **صفحه فرود (Landing Page) (`/var/www/openroute_website/`)**: فرانت‌اند ثابت HTML/CSS برای دامنه `openroute.ir` که فرم‌های پرداخت کریپتو و ریالی را در خود جای داده است.

## ⚙️ پیش‌نیازها

برای اجرای OpenRoute به موارد زیر نیاز دارید:
- Docker و Docker Compose
- توکن ربات تلگرام از [@BotFather](https://t.me/botfather)
- دیتابیس PostgreSQL و Redis (توسط Docker Compose مدیریت می‌شوند)
- (اختیاری) کلید API از [NOWPayments](https://nowpayments.io/) برای پرداخت‌های رمزارزی

## 🛠 نصب و استقرار

۱. **کلون کردن مخزن (Repository):**
   ```bash
   git clone https://github.com/AmirNaderi1997/OpenRoute.git
   cd OpenRoute/home/openroute/selling_vpn_bot
   ```

۲. **تنظیم متغیرهای محیطی (Environment Variables):**
   فایل `.env` را باز کرده و مقادیر مورد نیاز را با کلیدها و رمزهای واقعی خود جایگزین کنید.
   ```env
   BOT_TOKEN=YOUR_TELEGRAM_BOT_TOKEN_HERE
   POSTGRES_PASSWORD=YOUR_DB_PASSWORD
   NOWPAYMENTS_API_KEY=YOUR_NOWPAYMENTS_API_KEY
   # ... و سایر مواردی که نیاز به جایگزینی دارند
   ```

۳. **استقرار با استفاده از Docker Compose:**
   ```bash
   docker compose up -d --build
   ```

۴. **راه‌اندازی ربات:**
   - ربات خود را در تلگرام باز کنید (مثلاً `@getopenroutebot`).
   - دستور `/start` را ارسال کنید.
   - برای دسترسی به پنل مدیریت، دستور `/admin_login` را ارسال کرده و `SUPERADMIN_PASSWORD` تنظیم شده را وارد کنید.
   - در نهایت، دستور `/setup_topics` را در گروه مدیریت اختصاصی خود ارسال کنید تا ربات کانال‌های نوتیفیکیشن پرداخت‌ها را بشناسد.
